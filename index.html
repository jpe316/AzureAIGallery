MODEL Tiny YOLOv2 1.2 This model is a real-time neural network for object detection that detects 20 different classes. It is made up of 9 convolutional layers and 6 max-pooling layers and is a smaller version of the more complex full YOLOv2 network. ONNX Link: https://github.com/onnx/models/tree/master/tiny_yolov2
MODEL MNIST Handwritten Digit Recognition 1.2 This model predicts handwritten digits using a convolutional neural network (CNN) using MNIST (Modified National Institute of Standards and Technology) dataset. ONNX , WinML Link: https://github.com/onnx/models/tree/master/mnist
MODEL FER+ Emotion Recognition 1.2 This model is a deep convolutional neural network for emotion recognition in faces. ONNX , WinML Link: https://github.com/onnx/models/tree/master/emotion_ferplus
